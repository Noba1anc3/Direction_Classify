# 测试集实验结果

## 1. 测试标准
测试数据为三通道彩色图像，其标注为方向类别和具体的旋转角度；模型输出结果为预测出的回归角度。  
对实验一，如若回归角度与标注所在方向类别轴之间的角度差在45°以内，则认定为预测正确。  
对实验二，如若回归角度与标注所在方向类别轴之间的角度差在30°以内，则认定为预测正确。  
如上所述，实验二的标准比实验一更高。  

## 2. 总体实验结果

| 实验           | 准确率 |
|:-------------:|:------:|
| 三通道原图 30° |  58.0% |
| 三通道原图 45° |  64.3% |
